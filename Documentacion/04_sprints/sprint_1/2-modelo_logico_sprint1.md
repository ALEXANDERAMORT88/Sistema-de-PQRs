# Modelo Lógico – Sprint 1

El modelo lógico representa la estructura de la base de datos para soportar el registro de usuarios y la gestión inicial de PQRs.

## Tablas
- usuario
- pqr
- tipo_pqr

## Relaciones
- Un usuario puede registrar múltiples PQRs.
- Cada PQR está asociada a un tipo de PQR que define su tiempo máximo de respuesta.
